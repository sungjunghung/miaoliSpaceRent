## 共用符號建議

- 開始事件：接收申請、提交申請、排程觸發。
- 結束事件：成立、取消、退款完成、駁回結案。
- 任務：會員任務、承辦任務、會計任務、出納任務、系統任務。
- 閘道：是否需文件、是否逾期、是否核准、是否駁回。
- 中介事件：期限到期、通知發送、資料回補。
